vagrant
===

download
---
http://www.vagrantup.com/downloads

find boxes
---
https://atlas.hashicorp.com/boxes/search

add box
---
```bash
$ vagrant box add [box name]
```

create Vagrantfile
---
```bash
$ vagrant init [box name]
```

up and running
---
```bash
$ vagrant up
$ vagrant ssh
```
